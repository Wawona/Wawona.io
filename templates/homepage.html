Wawona is a native, high-performance Wayland compositor for macOS, iOS, and Android. Built on a shared Rust core with direct Metal and Vulkan rendering — zero-copy, bare-metal speed, no VMs.

Native Compositing

Wawona runs underneath the host OS's native window management layer — Quartz on macOS, UIKit on iOS, Android's WindowManager on Android. Wayland surfaces are promoted to first-class native windows with full system integration.

// Wayland surface → native macOS window
let surface = compositor.create_surface();
let window = quartz.promote_to_nswindow(surface);
window.make_key_and_order_front();

Zero-Copy Rendering

Direct Metal and Vulkan pipelines with zero-copy IOSurface integration. Client buffers are mapped directly to GPU-backed textures without intermediate copies — near-instant frame delivery at native refresh rates.

One Core, Three Platforms

A shared Rust library manages all compositor state, protocol dispatch, surface management, and IPC. Platform frontends connect via UniFFI and C FFI bindings, so every platform runs the exact same compositor logic.

Waypipe + Remote Apps

Run remote Linux GUI applications over SSH and display them natively on your device. Built-in Waypipe and Dropbear SSH on mobile platforms — no separate SSH client needed.

// SSH into remote Linux host
// Forward Wayland display
waypipe ssh user@server
