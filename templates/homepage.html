Wawona

A native, high-performance Wayland compositor for macOS, iOS, and Android.

Powered by a shared Rust core with direct Metal & Vulkan rendering. Zero-copy. Bare-metal speed. No VMs.

Get Started View Source

Why Wawona?

A compositor built from the ground up for the platforms you actually use.

🖥️

Native Compositing

Sits underneath Quartz on macOS, UIKit on iOS, and Android's WindowManager. Wayland surfaces become first-class native windows.

⚡

Bare-Metal Rendering

Direct Metal (Liquid Glass) and Vulkan pipelines with zero-copy IOSurface integration for near-instant frame delivery.

🦀

Rust Core

186+ crates managing compositor state, protocol dispatch, and surface management with memory safety. Cross-platform via UniFFI bindings.

🌐

68 Wayland Protocols

Registers 68 protocol globals including wl_compositor, xdg_wm_base, and wlr extensions. Growing implementation coverage every release.

🔧

Nix-Powered Builds

Hermetic, reproducible builds that cross-compile 27 native C/C++ libraries from source for macOS, iOS, and Android. Nix is your only dependency.

📡

Waypipe Integration

Run remote Linux GUI apps over SSH and display them natively on your device. Built-in Waypipe + Dropbear SSH on mobile platforms.
